# 0x02. Redis Basics
Introduction to Redis basics.  

## Definition 

Redis stands for __Remote Dictionary Server__. It is an:  
- Open-source  
- In-memory data structure store.   
It is primarily used as a:   
- Database,  
- Cache, and  
- Message broker.  
This is due to its:  
- High performance,   
- Flexibility, and  
- Wide range of data structures it supports.
